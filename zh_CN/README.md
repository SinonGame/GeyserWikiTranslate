_一座连接Minecraft: Bedrock Edition 和 Minecraft: Java Edition的桥梁_

**目前支持MC Bedrock v1.16.0/1 和 MC Java v1.16.1**

Geyser的目标是通过让Bedrock客户端加入Java版服务器来架通Minecraft: Bedrock版 和 Minecraft: Java版
